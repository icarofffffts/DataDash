# 📊 DataDash Project

> Inteligência Artificial aplicada ao análise de dados CSV.

### 🎯 Proposta
Uma interface moderna para visualização de métricas e geração de insights automáticos a partir de arquivos CSV, focada em simplicidade e poder analítico.

### 🛠️ Core Stack
- **Next.js 15**
- **Tailwind CSS 4**
- **Recharts**
- **Google Gemini SDK**

### 🌟 Destaques
- **Privacy-First**: Processamento de CSV via Client-Side.
- **AI-Powered**: Relatórios automatizados explicando seus dados.
- **High Performance**: Dashboard fluido com animações `motion`.
