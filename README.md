# 📊 DataDash - CSV Dashboard Analytics

[![Next.js](https://img.shields.io/badge/Next.js-15-black?style=flat-square&logo=next.js)](https://nextjs.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-4-38B2AC?style=flat-square&logo=tailwind-css)](https://tailwindcss.com/)
[![Gemini AI](https://img.shields.io/badge/Gemini_AI-Pro-blue?style=flat-square&logo=google-gemini)](https://ai.google.dev/)

**DataDash** é uma aplicação interativa projetada para transformar relatórios de atendimento e dados brutos em arquivos CSV em dashboards visuais e insights inteligentes utilizando inteligência artificial.

## ✨ Funcionalidades

- 📁 **Upload de CSV**: Interface intuitiva para carregamento de arquivos de dados.
- 📊 **Dashboards Dinâmicos**: Visualização instantânea de métricas usando Recharts.
- 🤖 **Insights com IA**: Integração com Google Gemini para análise de tendências e sugestões automáticas.
- 🎨 **Interface Premium**: Design moderno, responsivo e focado na experiência do usuário (UX).

## 🚀 Tecnologias

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Estilização**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Gráficos**: [Recharts](https://recharts.org/)
- **IA**: [Google Gemini SDK](https://ai.google.dev/)
- **Parsing**: [PapaParse](https://www.papaparse.org/)

## 🛠️ Instalação e Execução

### Pré-requisitos
- Node.js 20+
- NPM ou Yarn

### Passo a Passo

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/icarofffffts/DataDash.git
   cd DataDash
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Configure as variáveis de ambiente:**
   Crie um arquivo `.env.local` na raiz do projeto:
   ```env
   GEMINI_API_KEY="sua_chave_api_aqui"
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```
   Acesse `http://localhost:3000` no seu navegador.
